# 🛡️ Network Intrusion Detection System

A machine learning classification project that detects and classifies malicious network traffic using the UNSW-NB15 cybersecurity dataset.

## 🚀 Features

- Data cleaning and preprocessing
- Categorical feature encoding using OrdinalEncoder
- Class imbalance handling using sample weights
- Random Forest and XGBoost classifiers
- Feature importance analysis
- Multi-class attack classification
- Model evaluation using F1 Score and Classification Report

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
